These methods help you handle multiple promises or control asynchronous flow more effectively:

Method Purpose
Promise.all() Waits for all promises to fulfill; rejects if any fail.
Promise.allSettled() Waits for all to settle (fulfilled or rejected); returns all results.
Promise.race() Resolves/rejects as soon as the first promise settles.
Promise.any() Resolves with the first fulfilled promise; rejects only if all fail.
Promise.resolve() Returns a promise that is already fulfilled with a given value.
Promise.reject() Returns a promise that is already rejected with a given reason.
Promise.finally() Executes a callback regardless of fulfillment or rejection.

Await Keyword in JavaScript

The await keyword is used to pause execution inside an async function until a Promise is settled.
It makes asynchronous code look and behave like synchronous code, improving readability and flow.

ðŸ“‹ await Usage Summary

Feature Purpose
await promise Pauses execution until the promise is fulfilled or rejected.
await non-promise Wraps the value in a resolved promise and waits for it.
await Promise.reject() Throws the rejection reason as an error.
await Promise.resolve() Returns the resolved value directly.
await Promise.all() Waits for all promises to resolve; throws if any reject.
await Promise.race() Waits for the first promise to settle (resolve or reject).
await Promise.any() Waits for the first fulfilled promise; throws if all reject.
await Promise.allSettled() Waits for all promises to settle and returns their results.
